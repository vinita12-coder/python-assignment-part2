# Part 2: Data Structures — Restaurant Order Management System

A Restaurant Order Management System built using Python's core data structures.

## Tasks
- **Task 1:** Explore the Menu — grouped display, stats, most expensive item
- **Task 2:** Cart Operations — add, remove, update quantity, order summary with GST
- **Task 3:** Inventory Tracker — deep copy demo, stock deduction, reorder alerts
- **Task 4:** Sales Log Analysis — revenue per day, best-selling day, most ordered item

## Concepts Used
- Lists, Dictionaries, Nested data structures
- Dictionary methods: keys, values, items
- List comprehensions and filtering
- deep copy vs shallow copy
- enumerate, lambda, max with key

## How to Run
```bash
python part2_order_system.py
```
